WHAT AIONIQ IS

AiONIQ Labs is a hybrid digital studio + intelligent product environment.
It blends:
	•	AI engineering
	•	Brand systems
	•	Design excellence
	•	High-end web-app UI
	•	Portfolio-level visual storytelling

The website behaves like a Single Page Application (SPA) but feels like a premium design portfolio.
A union of Apple × Stripe × OpenAI × Linear — clean, intelligent, futuristic.

⸻

WHAT THE SITE MUST ACHIEVE

The AiONIQ site needs to be:
	•	Visually premium
	•	Functionally modern
	•	Smooth like a web-app
	•	Scalable like a product
	•	Fast and minimal
	•	Carefully animated
	•	Responsive up to 1800px

Content must be clear, structured, and intentional.

⸻

TECH STACK

All suggestions, components, or code should assume:
	•	Next.js App Router
	•	TailwindCSS
	•	shadcn/ui
	•	Framer Motion
	•	Optional Spline 3D (only for meaningful visuals)
	•	Responsive mobile-first design
	•	Typed React components (TS)

⸻

LAYOUT & CONTAINER STRUCTURE

AiONIQ uses a multi-container system, not one fixed max-width.

1. Marketing/Text Sections
	•	max-width: 1280px
	•	Best for paragraphs, stories, service explanations
	•	Keep line length ± 55–75 characters

2. Visual/Bento Sections
	•	max-width: 1440px
	•	Used for feature blocks, bento cards, visual storytelling

3. Showcase/Hero Sections
	•	max-width: 1600px → 1728px
	•	Used for animated hero, 3D sections, large visuals

4. App Mode (future dashboard)
	•	max-width: up to 1800px
	•	Only used for product-like UI

Padding rules
	•	Desktop: 40–48px
	•	Tablet: 24–32px
	•	Mobile: 16–20px

⸻

ANIMATION RULES (Framer Motion)

AiONIQ uses micro-interactions, not dramatic animations.
	•	Easing: cubic-bezier(.2,.9,.22,1)
	•	Subtle hover lift (4–6px)
	•	Smooth section fade-in
	•	Bento grids use staggered reveal
	•	No bouncy or cartoonish effects
	•	Mobile animations must be softer

⸻

3D & VISUALS

AiONIQ supports 3D ONLY when it adds clarity or emotional value, not decoration.

Examples:
	•	Hero
	•	Feature spotlight
	•	Interaction concept visual

Always require:
	•	Mobile fallback
	•	Lazy loading
	•	No blocking scripts
